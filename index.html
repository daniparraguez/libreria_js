FormValidation


Librería FormValidation

FormValidation es una librería para la validación de los campos de formulario por parte del cliente. FormValidation es para verificar que las entradas son del tipo correcto y que estén completas con datos, antes de que el formulario sea enviado. Los campos de formulario son una tarea de programación necesaria, aunque muchas veces es un proceso molesto. En javascript facilitan la programación y mejora la interfaz del usuario. Esto te ahorrará una tarea importante.

Instalación

Uso

Despues de la instalación, crear un formulario o dos input con id respectivo mas un boton (como el ejemplo), en nuestro documento HTML

Primero, enlace javascript en la página.

Ahora, adjunte validateUser al campo de texto.

Ahora, adjunte validatePassword al campo de texto.

Uso en Node.js

Test

FormValidation se prueba con Mocha. Ejecutar mocha test-name.js o mocha test-password.js desde la línea de comando para ejecutar el banco de pruebas.
